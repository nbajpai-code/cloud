# OCI Architect & Developer Guide

## 1. Cloud Architecture Frameworks & Patterns
Resources for building on Oracle Cloud Infrastructure.

- **OCI Architecture Center**
    - Reference architectures, solution playbooks, and best practices.
    - [Architecture Center](https://docs.oracle.com/en/solutions/)
- **OCI Well-Architected Framework**
    - *Pillars*: Security & Compliance, Reliability & Resilience, Performance & Cost Optimization, Operational Efficiency.
- **OCI Landing Zones**
    - Templates for secure and scalable cloud environments.

## 2. Developer Tools & SDKs
Tools for building and integrating with OCI.

- **OCI SDKs**
    - Libraries for Java, Python, TypeScript, .NET, Go, Ruby.
    - Open-source and available on GitHub.
    - [SDKs and CLI](https://docs.oracle.com/en-us/iaas/Content/API/Concepts/sdks.htm)
- **OCI CLI**
    - Command-line interface for all OCI services.
- **DevOps Services**
    - Built-in tools for CI/CD, code repositories, and deployment.

## 3. Certifications & Learning Paths
- **OCI Architect (Associate/Professional)**
    - *Focus*: Planning, designing, implementing, and operating OCI solutions.
- **OCI Developer (Associate/Professional)**
    - *Focus*: Developing cloud-native applications on OCI.
- **Learning Resources**
    - [Oracle University (Free Tier included)](https://education.oracle.com/)
    - [OCI Developer Guide](https://docs.oracle.com/en-us/iaas/Content/Developer/Concepts/devguides.htm)

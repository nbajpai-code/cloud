# OpenStack Private Cloud Guide

## 1. Architecture & Components
Resources for designing open standard cloud computing platforms.

- **Core Components**
    - **Nova**: Compute (manage instances).
    - **Neutron**: Networking (networks, subnets, routers).
    - **Cinder**: Block Storage.
    - **Swift**: Object Storage.
    - **Keystone**: Identity service.
    - **Glance**: Image service.
- **OpenStack Architecture Design Guide**
    - [Design Guide](https://docs.openstack.org/arch-design/)

## 2. Developer Tools & CLI
Interaction with OpenStack clouds.

- **OpenStack Client (CLI)**
    - Unified command-line client (`openstack`).
    - *Common Commands*:
        - `openstack server list`
        - `openstack network create`
        - `openstack image list`
    - [CLI Documentation](https://docs.openstack.org/python-openstackclient/latest/)
- **Python SDK**
    - `python-openstackclient` and library for building applications.
- **Heat Orchestration**
    - Template-based orchestration for describing cloud applications.

## 3. Certifications & Learning
- **Certified OpenStack Administrator (COA)**
    - *Focus*: Daily operation and management of an OpenStack cloud.
- **Learning Resources**
    - [OpenStack Docs](https://docs.openstack.org/)
    - [OpenStack Marketplace (Training)](https://www.openstack.org/marketplace/training/)

# VMware Private Cloud Guide

## 1. Architecture & Frameworks
Resources for designing Software-Defined Data Centers (SDDC) with VMware.

- **VMware vSphere**
    - The core virtualization platform (ESXi hypervisor + vCenter Server).
    - [vSphere Documentation](https://docs.vmware.com/en/VMware-vSphere/index.html)
- **VMware vCloud Director (VCD)**
    - Cloud service delivery platform for service providers.
- **vCloud Architecture Toolkit (vCAT)**
    - Technical and operational guidance for designing hybrid cloud solutions.
- **VMware Validated Solutions**
    - Technical designs for building secure, compliant, and resilient infrastructure.

## 2. Developer Tools & APIs
Tools for automating and building on VMware.

- **PowerCLI**
    - Powerful command-line tool based on PowerShell for managing vSphere and vSAN.
    - [PowerCLI Documentation](https://developer.vmware.com/powercli)
- **vSphere Web Services SDK**
    - APIs for managing virtual machines, datacenters, and networks.
- **vCloud API**
    - RESTful API for building interactive clients for vCloud Director.
- **Vmware {code}**
    - Developer hub with samples and SDKs. [developer.vmware.com](https://developer.vmware.com/)

## 3. Certifications & Learning
- **VMware Certified Professional (VCP-DCV)**
    - *Focus*: Data Center Virtualization regarding vSphere infrastructure.
- **VMware Certified Advanced Professional (VCAP)**
    - Design and Deploy tracks for advanced architects.
- **Learning Resources**
    - [VMware Learning](https://www.vmware.com/learning.html)
    - [VMware Hands-on Labs (HOL)](https://hol.vmware.com/)

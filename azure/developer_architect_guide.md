# Azure Architect & Developer Guide

## 1. Cloud Architecture Frameworks & Patterns
Guidance for designing solutions on Microsoft Azure.

- **Azure Architecture Center**
    - The hub for architectural guidance, reference architectures, and design patterns.
    - [Architecture Center](https://learn.microsoft.com/en-us/azure/architecture/)
- **Azure Well-Architected Framework**
    - *Pillars*: Cost Optimization, Operational Excellence, Performance Efficiency, Reliability, Security.
    - [Well-Architected Review](https://learn.microsoft.com/en-us/assessments/)
- **Cloud Adoption Framework**
    - Best practices for cloud adoption and governance.

## 2. Developer Tools & SDKs
Tools for developing applications on Azure.

- **Azure SDKs**
    - Libraries for .NET, Java, Python, JavaScript/TypeScript, Go.
    - Designed to be idiomatic and consistent.
    - [Azure SDKs](https://azure.microsoft.com/en-us/downloads/)
- **Azure CLI & PowerShell**
    - Command-line tools for managing resources.
- **Visual Studio & VS Code Integration**
    - Deep integration for deploying and debugging Azure applications.

## 3. Certifications & Learning Paths
- **Azure Solutions Architect Expert**
    - *Focus*: Designing cloud and hybrid solutions on Azure.
- **Azure Developer Associate**
    - *Focus*: Designing, building, testing, and maintaining cloud applications.
- **Learning Resources**
    - [Microsoft Learn](https://learn.microsoft.com/)
    - [Azure Developer Center](https://developer.microsoft.com/en-us/azure/)

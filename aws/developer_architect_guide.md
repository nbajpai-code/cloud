# AWS Architect & Developer Guide

## 1. Cloud Architecture Frameworks & Patterns
Resources for designing robust systems on Amazon Web Services.

- **AWS Well-Architected Framework**
    - *6 Pillars*: Operational Excellence, Security, Reliability, Performance Efficiency, Cost Optimization, Sustainability.
    - [Well-Architected Tool](https://aws.amazon.com/well-architected-tool/)
    - [Documentation](https://aws.amazon.com/architecture/well-architected/)
- **AWS Architecture Center**
    - Reference diagrams, vetted solutions, and architectural best practices.
    - [Architecture Center](https://aws.amazon.com/architecture/)
- **AWS Solutions Library**
    - Implementation guidance for common business and technical use cases.

## 2. Developer Tools & SDKs
Essential tools for building on AWS.

- **AWS SDKs**
    - Libraries for Python (Boto3), JavaScript, Java, .NET, Go, etc.
    - [Tools to Build on AWS](https://aws.amazon.com/developer/tools/)
- **AWS Command Line Interface (CLI)**
    - Unified tool to manage AWS services.
- **AWS Cloud Development Kit (CDK)**
    - Define cloud infrastructure using familiar programming languages.
    - [AWS CDK](https://aws.amazon.com/cdk/)
- **AWS Builder Center**
    - Tutorials and code samples.

## 3. Certifications & Learning Paths
- **AWS Certified Solutions Architect (Associate/Professional)**
    - *Focus*: Designing distributed systems on AWS.
- **AWS Certified Developer (Associate)**
    - *Focus*: Developing and maintaining applications on AWS.
- **Learning Resources**
    - [AWS Skill Builder](https://skillbuilder.aws/)
    - [AWS Developer Documentation](https://docs.aws.amazon.com/)

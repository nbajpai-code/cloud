# GCP Architect & Developer Guide

## 1. Cloud Architecture Frameworks & Patterns
For Cloud Architects designing on Google Cloud, these resources are essential for building secure, scalable, and resilient solutions.

- **Google Cloud Architecture Framework (GCAF)**
    - *Core Pillars*: Operational Excellence, Security, Reliability, Performance Optimization, Cost Optimization.
    - [GCAF Documentation](https://cloud.google.com/architecture/framework)
- **Google Cloud Well-Architected Framework**
    - A subset of GCAF focusing on assessment and improvement.
- **Reference Architectures**
    - Blueprints for common workloads (Web Apps, Data Analytics, AI/ML).
    - [Architecture Center](https://cloud.google.com/architecture)
- **Cloud Adoption Framework**
    - Strategy for organizational cloud transformation.

## 2. Developer Tools & SDKs
For Cloud Developers building applications on GCP.

- **Google Cloud SDK**
    - Includes `gcloud`, `bq` (BigQuery), `gsutil` (Storage).
    - [Install SDK](https://cloud.google.com/sdk/docs/install)
- **Client Libraries (SDKs)**
    - Idiomatic libraries for Java, Python, Node.js, Go, etc.
    - [Client Libraries](https://cloud.google.com/apis/docs/cloud-client-libraries)
- **Cloud Code**
    - IDE extensions for VS Code and IntelliJ to develop and deploy to Kubernetes/Cloud Run.
- **Cloud Shell**
    - Browser-based terminal with pre-installed tools.

## 3. Certifications & Learning Paths
- **Professional Cloud Architect**
    - *Focus*: Design, manage, and secure robust cloud solutions.
    - [Exam Guide](https://cloud.google.com/learn/certification/cloud-architect)
- **Professional Cloud Developer**
    - *Focus*: Build scalable and highly available applications using recommended practices.
    - [Exam Guide](https://cloud.google.com/learn/certification/cloud-developer)
- **Learning Resources**
    - [Google Cloud Skills Boost](https://www.cloudskillsboost.google/)
    - [Google Cloud Developer Center](https://cloud.google.com/developers)
